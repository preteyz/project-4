Welcome to Venture!

A website that conatins collection of the most popular vacation spots on the browse page. If your favorite location doesn't exist, it can be added directly from the browse page by a logged-in user. Each travel location includes an environment type so people can filter travel locations by what environment they prefer during their vacation (beach, mountains, parks, cities).

If you already know the location you wish to read reviews for, the search bar can be used to filter by both environment type and location name. Users can write reviews and leave ratings which will influence which location is displayed first. Users can also favorite locations to be displayed on their profile page.
